# Programacion-Actividad-2.3
Realizaremos una practica para saber utlizar GitHub

## Objetivo
Desarrollar un proyecto colaborativo usando GitHub aplicando
los conceptos de programación estructurada.

## Requisitos
- Cuenta en GitHub
- Navegador web
- Conocimientos básicos de programación

## Instrucciones de ejecución
1. Clonar el repositorio
2. Abrir los archivos del proyecto
3. Ejecutar el programa según las indicaciones
